Summary

A summary of feeback given on our website, and the changes we implemented to improve.

  • Text

    Lots of text on some pages, to fix this we added dropdowns for large text portions.

  • Interactivity

    The interactivity was good, but sparse throught website. To fix this we added more engaging portions, for example the image carousel.

  • Members

    It was hard to find more information about our team memebers, so we added links to our social medias.

  • Styling

    Website feeback indicated we could improve ours styling so we added better colors and elements to containers to make them more appealing
